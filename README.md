# Лабораторная работа №1: Арифметические выражения на ассемблере

## Вариант №87

### Выражение

(a³ + b³) / (a² * c - b² * d + e)


### Размеры переменных (беззнаковые)

| Переменная | Размер |
|------------|--------|
| a | 64 бит |
| b | 32 бит |
| c | 32 бит |
| d | 8 бит |
| e | 16 бит |

### Требования

- Переменные и результат — в области инициализированных данных
- Использовать операнды наименьшего возможного размера
- Обработка исключительных ситуаций (деление на ноль, переполнение)

### Коды возврата

| Код | Описание |
|-----|----------|
| 0 | Успех, результат сохранён |
| ≠0 | Ошибка, результат не сохранён |

### Сборка и запуск (NASM)

```bash
nasm -f elf64 lab1.asm -o lab1.o
ld lab1.o -o lab1
./lab1
echo $?   # код возврата
